我的Git自学笔记

常用的Git指令：

  • git status: 显示当前库的状态。

  • git add <file>: 将文件加入暂存区。

  • git commit: 提交。

    • -m：输入提交的信息。
    • –amend：修改提交的信息。

  • git log: 查看提交日志。

    • -p：查看提交所带来的变动。
    • –**graph:**以图标的形式查看分支。

  • **git push:**更新GitHub上的仓库。

  • **git init:**初始化仓库。

  • **git diff:**查看更改前后的差别。

    • **HEAD:**查看与最新提交的差别。

  • **git branch:**显示分支一览表。

  • git checkout -b filename：创建新的分支并切换到该分支。

  • git merge –no-ff branchname：合并分支。

  • **git reset –hard :**回溯命令。

  • **git reflog:**查看当前仓库的操作日志。

  • git rebase -i：压缩历史。

  • **git remote add:**添加远程仓库。

  • **git push -u origin master:**推送至master分支。
